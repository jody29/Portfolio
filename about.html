Wie ben ik?

Mijn naam is Jody Lorist en ik ben een UX/UI designer en Frontend Developer.

Ik zou mij zelf omschrijven als een heel erg onderzoekend en empatisch persoon. Ik probeer altijd wel alles tot de bodem uit te zoeken. Deze eigenschappen helpen mij enorm voor zowel UX/UI design als frontend development.

Ik kan erg goed samenwerken in een multi-disciplinair team, maar zelfstandig aan een project werken is voor mij ook geen probleem. Zelf heb ik eigenlijk geen voorkeur.

Zelf kan ik altijd wel heel erg gefocused aan een project werken, maar natuurlijk ben ik altijd wel in voor gezelligheid op de werkvloer.

Ben je nou benieuwd geworden naar mij? Neem dan vooral contact met mij op!

Vaardigheden

Frontend Development

Hierin heb ik mij het meeste ontwikkeld tijdens mijn eerste 2 jaar op CMD. Voordat ik op CMD kwam had ik eigenlijk helemaal geen ervaring met ook maar een codeertaal, maar ik pakte het eigenlijk heel snel op. Ik ben nu erg ervaren met HTML, CSS en JavaScript. Ik heb een basis kennis van backend talen zoals PHP en MySQL. Ik leer uiteraard nog elke dag. Zo ben ik bijna iedere dag wel bezig met cursussen volgen op Codecademy.

UX/UI Design

Dit is toch wel een vaardigheid van mij die ik later pas ben gaan ontdekken. Dit is vooral door feedback van docenten. Ik ben namelijk erg goed in empatisch denken en ben erg goed in onderzoek doen naar verschillende User Interface Stijlen. Hierdoor kan ik heel goed uitkomen op een user ervaring die goed werkt voor de doelgroepen en zo zal het interface er altijd erg goed uitzien!.

Adobe CC en andere design software

Door mijn voorgaande MBO opleiding 'Media Design', had ik al een hele hoop ervaring met de Adobe CC software. Ik beheerste software zoals Illustrator, Photoshop, Premiere Pro en After Effects al erg goed. Bij CMD is Adobe XD er voor mij bijgekomen. Ook heb ik voor een project gewerkt met sketch. Hier ben ik ook al aardig bekend mee geworden.

Video editing

Ik heb door de opleiding 'Media Design' en diverse stages (Video-redacteur bij XGN en Stagiare-video bij IJburgTV) erg veel ervaring opgedaan met video editing. Dit doe ik vooral in programma's zoals Premiere Pro en Final Cut Pro. Ook kan ik erg goed omgaan met camera's en heb ik aardig wat affiniteit met filmen en fotograferen.
