Website AFC Ajax

Opdrachtgever: eigen project

Focus: Frontend Development (Responsive & Surface)

Datum: 18-10-2020

Over dit project

Voor het vak Frontend Development was het de bedoeling dat we een site naar keuze gingen nabouwen doormiddel van HTML, CSS en JavaScript. Ik heb ervoor gekozen om de website van AFC Ajax na te bouwen. We mochten zelf kiezen of we de focus gingen leggen op responsivenes of de surface laag. Ik heb ervoor gekozen om voor beide te gaan. Dit is omdat ik mijzelf graag wilde uitdagen met coderen.

Ik ben begonnen om de website eerst voor mobiel goed werkend te maken, daarna ben ik pas gaan kijken hoe ik de website responsive ben gaan maken. Uiteraard heb ik bij het coderen wel eraan gedacht om zoveel mogelijk css te gebruiken die uiteindelijk ervoor zorgen dat het responsive maken veel makkelijker gaat worden. Voor deze opdracht heb ik als eindcijfer een 8 gekregen.
