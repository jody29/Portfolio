Customize Playstation

Opdrachtgever: eigen project

Focus: Frontend Development (JavaScript)

Datum: 25-05-2020

Over dit project

Bij dit project was het de bedoeling om een site te bouwen met HTML, CSS en JavaScript. De focus lag hier vooral op het javascript gedeelte. Ik heb een website gemaakt waar de gebruiker zijn eigen Playstation 4 console kan customizen, aan zijn winkelwagen kan toevoegen en vervolgens ook op afrekenen kan klikken. Wanneer de gebruiker op afrekenen klikt krijgt een een melding waarin hij/zij wordt bedankt voor de betaling en kan vervolgens op het kruisje klikken om terug te gaan naar de default versie van de playstation.

Dit project was voor het vak Inleiding Programmeren. Dit vak had ik in het eerste jaar van CMD. Ik heb deze opdracht vervroegd mogen afronden, omdat ik heel erg vooruit liep met alles. Ik heb voor dit project dan ook een 9 gekregen. Dit cijfer is gebasseerd op mijn toepassing van JavaScript.
